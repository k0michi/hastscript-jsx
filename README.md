# hastscript-jsx

A library to create HAST trees with JSX.

## Example

```jsx
const tree = (
  <div class="foo" id="some-id">
    <span>some text</span>
    <input type="text" value="foo" />
    <a className="alpha bravo charlie" download>deltaecho</a>
  </div>
);
```

## Installation

```
yarn add hastscript-jsx
```
